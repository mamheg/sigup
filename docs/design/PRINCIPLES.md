# SiGup — дизайн-принципы

Живой свод. Один взгляд на экран должен читаться как «дорого и продуманно».

## Визуальный тезис
Тёплый премиальный маркетплейс Северного Кавказа: воздух и типографика бутика + плотность и продукт-форвардность большого маркета. Зелёно-песочная палитра, золото — редкий акцент.

## Токены (истина — `src/index.css @theme`)

**Типографика**
- Cormorant Garamond — только display ≥24px (H1–H3, hero, акцент цены). Weight 600, трекинг `--tracking-display` (−0.02em) на крупном.
- Inter — всё ≤20px. Body 16/1.5, вторичное 14, подпись 12. Weights 400/500/600.
- `text-balance` на заголовках, `text-pretty` на body (дефолт), `.tabular` на ценах/метриках.

**Спейсинг / сетка**
- База 4px, шаг 8px. Ритм секций 64–96px desktop / 32–48px mobile.
- Каталог-грид: 2→3→4→5→6 колонок (480/640/1024/1280/1536), гэп 12–16px.
- Max-width: каталог 1440–1536, контент 1200.

**Цвет / радиусы**
- Бренд `brand #244D33`; акцент `gold #C79E61` — только CTA/цена/бейджи.
- Нейтраль: `canvas #FBFAF7`, `surface #fff`, `ink/ink-soft/ink-faint`, границы `line/line-strong`.
- Радиусы: `sm 8` (инпуты/кнопки) · `md 12` (карточки) · `lg 16` (панели) · `xl 22` (hero) · full (аватары/чипы). Concentric: outer = inner + padding.

**Тени (3 тира, зелёный подтон)**
- `shadow-sm` микролифт · `shadow-card` карточки/дропдауны · `shadow-pop` модалки/поповеры.
- Тень — для лифта, не для разделения. Тест: карточка организована и без тени (граница+спейсинг).

**Движение**
- Длительности: press 120ms (`scale 0.96`), tooltip 150, dropdown 200, modal 300. Потолок ≤300ms.
- Изинги: `--ease-smooth` (UI), `--ease-drawer` (шиты). Пружины motion: `bounce:0` для UI.
- Вход: opacity0 + y12 + blur4 → 0; стаггер 50–80ms. Hover-lift `-4px` только под `hover:hover`. Уважать `prefers-reduced-motion`.

## Do / Don’t
- ✅ Собирать экраны из `ui/` + `catalog/` + `patterns/`; не изобретать стиль на месте.
- ✅ Каждый экран: default/hover/focus/disabled/loading(skeleton)/empty/error.
- ✅ Реальные фото 1:1 с image-outline `black/10`; читаемый контраст (AA); видимый focus.
- ❌ Ad-hoc hex в компонентах, разнобой радиусов/теней, `transition: all`, градиенты-заглушки, «ИИ-копирайт» в UI.

## Не выглядит как ИИ (litmus)
Бренд узнаётся на первом экране? Один сильный якорь? Понятно по заголовкам? У каждой секции одна задача? Карточки правда нужны там, где стоят? Движение улучшает иерархию? Осталось бы «дорого» без декоративных теней? Копирайт звучит как продукт, не как промпт?
